# Groq Meeting Assistant

ブラウザで動作する、高速・高精度なリアルタイム文字起こし & 会議分析ツールです。
Groq API (Whisper-large-v3) を使用して、超高速な文字起こしを実現しています。

## 特徴

- ⚡ **超高速文字起こし**: Groq CloudのWhisper-large-v3を使用
- 🎙️ **マルチソース対応**: マイク入力、システム音声（画面共有経由）、またはその両方を同時に録音可能
- ✨ **AI分析**: 会議終了後にLlama 3 (70B/8B) 等を使って要約やアクションアイテムを自動抽出
- 📱 **レスポンシブ**: PCおよびスマホでの快適な利用が可能
- 🔒 **プライバシー重視**: 音声データはブラウザから直接Groq APIに送信され、サーバーには保存されません。APIキーはローカルストレージにのみ保存されます。

## 使い方

1. このリポジトリをクローンまたはダウンロードし、`index.html`をブラウザで開きます。
2. 初回起動時に**設定（Settings）**画面が開きます。
3. [Groq Cloud Console](https://console.groq.com/keys) で取得したAPIキーを入力して保存してください。
   - APIキーはブラウザのローカルストレージに保存され、外部に送信されることはありません。
4. 「録音開始」ボタンを押して会議の文字起こしを開始します。

## 注意事項

- **APIキーについて**: ソースコードにはAPIキーは含まれていません。ご自身で発行したGroq APIキーが必要です。
- **システム音声の録音**: ブラウザのセキュリティ制約上、「システム音声」を録音する場合は、録音開始時に表示される画面共有の選択で「システムオーディオを共有」にチェックを入れる必要があります。
